# Limbus News Archive
2
A clean, searchable, and accessible archive of official Limbus Company notices.

This project is live at: [lcna.whosmalikx.com](https://lcna.whosmalikx.com)

## The Problem This Solves

The official Limbus Company notices on Steam are a great central resource, but they are published primarily as images. This makes it impossible to use standard search tools like `Ctrl+F` or a search bar to find specific information about past updates, events, or bug fixes.

I built this archive to solve that problem. I transcribed the notices into text and created a simple interface, making the entire history of announcements fully searchable and accessible.

## Features

*   **Fully Searchable:** Instantly search through every notice title and content body for keywords.
*   **Chronologically Ordered:** All notices are sorted from newest to oldest for easy browsing.
*   **Clean & Readable:** A minimal interface designed for a clear and focused reading experience.
*   **Direct Linking:** Every notice has its own URL, making it easy to share specific announcements.

## How It's Built

This is a lightweight static website built with plain HTML, CSS, and vanilla JavaScript. There's no complex framework, which keeps it fast and easy to maintain.

The core of the archive is a single `notices.json` file. This file is not manually edited; it is generated automatically by my separate [limbus-notice-pipeline](https://github.com/ItsMalikx/limbus-notice-pipeline) repository. That pipeline is responsible for transcribing, sorting, and structuring the data before it's deployed here.

## File Structure

The project has a straightforward structure:

*   **`index.html`**: The main landing page that displays the list of all notices.
*   **`notice.html`**: The template page used by the JavaScript to display a single, selected notice.
*   **`data/notices.json`**: The complete, structured dataset containing all notice information. This is the only file that needs to be updated to add new content to the site.
*   **`assets/`**: Contains all static files:
    *   `css/styles.css`: All the styling for the website.
    *   `js/`: The vanilla JavaScript modules that power the site.
        *   `main.js`: Loads the notice list on the homepage.
        *   `notice.js`: Loads a single notice's content.
        *   `search.js`: Handles the search functionality.
        *   `utils.js`: Contains helper functions used by other scripts.
*   **`sitemap.xml`**: Generated by the pipeline for search engine optimization.

## How I Update the Site

1.  When new notices are released, I run my `limbus-notice-pipeline`.
2.  The pipeline generates an updated `notices.json` and `sitemap.xml`.
3.  I copy the new versions of those two files into this repository.
4.  I commit the changes and deploy the updated static site.
